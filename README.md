# minesweeper
This is an implementation of Minesweeper written in C++ and Qt

Author: Fernando B. Giannasi

# Requirements
C++11

Qt 5

# Instructions to compile and Run
1. Linux:
 * cd minesweeper
 * qmake-qt5
 * make
 * ./minesweeper

2. Windows:
 * cd minesweeper
 * qmake
 * make -f Makefile.Release
